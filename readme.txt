Banregio Checkout Direct allow to recive payments with a membership in WooCommerce sites.

== Description ==

Banregio Checkout Direct for WooCommerce send request to the payment hub,  for clients with a membership number.


Major features in Akismet include:

* Basic https  client for payment hub.


== Installation ==

Es necesario tener instalado WooCommerce para poder utilizar este plugin, despues de instalar el plugin de Banregio y
activarlo ve a la seccion de pagos en la configuracion de WooCommerce introduce tu numero de afiliacion de prueba
y realiza las pruebas necesarias, cuando estes listo pon tu numero de afiliacion productivo y haz check en
"Habilita ambiente de producción"
